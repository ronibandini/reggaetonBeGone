# Connections
Oled SDA ->  Rpi GPIO 2
Oled SCL -> Rpi GPIO 3
Oled VCC -> Rpi VCC
Oled GND -> Rpi GND

Button pin 1 -> GPIO26
Button pin 2 -> GND

Power supply: 5V 3A
